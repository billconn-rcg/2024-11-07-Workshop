The Bash Shell

Bash is a commonly-used shell that gives you the power to do tasks more quickly.
Click your OS below for instructions specific to that OS.

  • Windows
  • MacOS
  • Linux
  1. Download the FREE Home Edition MobaXTerm installer package.
    Note: you can use the portable version instead if you want, you will need to skip the install steps and instead extract the zip file to use the program.
  2. Run the installer and follow the steps below:
    1. Click Next.
    2. Check the box to accept the License Agreement and then Click Next.
    3. Don't change the install location unless you have a good reason to.
    4. Click Install.
    5. Click Finish.
  3. Run MobaXTerm by clicking on it in the Start Menu and follow the steps below:
    1. In the top menu, click Settings.
    2. Click Configuration.
    3. Ensure that a persistent home and root directory are setup. They should look like the following
      Persistent home directory: _AppDataDir_\MobaXterm\home
      Persistent root (/) directory: _AppDataDir_\MobaXterm\root
    4. Click OK.
  4. Click 'Start Local Terminal'
  5. Now we will install nano and groff:
    1. Type apt install nano groff and press Enter.
    2. Type y to accept the terms of use and press Enter.
    3. Type y to download and install packages, then press Enter.

This will provide you with MobaXterm, nano, ssh, ssh-keygen, and ssh-copy-id which we will use in this workshop.

The default shell in some versions of macOS is Bash, and Bash is available in all versions, so no need to install anything. You access Bash from the Terminal (found in /Applications/Utilities). You may want to keep Terminal in your dock for this workshop.

To see if your default shell is Bash type echo $SHELL in Terminal and press the Return key. If the message printed does not end with '/bash' then your default is something else and you can run Bash by typing bash

If you want to change your default shell, see this Apple Support article and follow the instructions on "How to change your default shell".

In order to access remote X window programs, you will need an X Display server on your Mac. There is a free one called XQuartz located here. Please download and setup this software on your Mac if you wish to follow along with the GUI portion of our training.
